# @preliquify/core

Core primitives and utilities for building Liquid templates with JSX.

For documentation, see the [main Preliquify README](https://github.com/MichaelNusair/preliquify#readme).

## Installation

```bash
npm install @preliquify/core
# or
pnpm add @preliquify/core
# or
yarn add @preliquify/core
```

## Usage

```tsx
import { For, Conditional, $ } from '@preliquify/core';
```

For complete documentation, examples, and guides, please see the [main repository](https://github.com/MichaelNusair/preliquify).
